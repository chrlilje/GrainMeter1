Grain Measure App - Version 1

Minimal scaffold for a PySide6 desktop app to calibrate and manually measure grain sizes.

Quick start

1. Create a virtual environment with Python 3.14.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the app from the project folder:

```bash
python -m grain_measure_app.main
```

Notes
- Place your sample images in the `sampleimages` folder and use the toolbar to load them.
- This scaffold implements the UI and basic models; calibration and measurement flows are stubs ready to be implemented next.
